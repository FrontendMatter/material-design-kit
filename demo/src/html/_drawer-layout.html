Drawer Layout

A wrapper element that positions a Drawer and other content.

  • Basic drawer layout demo
  • Push drawer layout demo - Enables a push effect on the layout content when toggling the drawer
  • Right drawer layout demo - Aligns the drawer on the right
  • Force narrow drawer layout demo - Ignores the responsiveWidth option and uses the narrow layout on any screen size
  • Drawer layout with header layout demo - Uses a Header Layout with a custom scrolling region
  • Drawer layout with custom scrolling region - Uses a custom scrolling region instead of the document
